# 람다식

## 사용 방법

1. Visual Studio에서 `Chapter41-Lambda.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 교안 코드 따라치기 - 기본 람다식, 문장 람다 (5개 과제) |
| CodingPractice-02 | 교안 코드 따라치기 - Func, Action, 익명 메서드 비교 (3개 과제) |
| CodingPractice-03 | 교안 코드 따라치기 - 람다식 매개변수, Predicate, 클로저 (5개 과제) |
| CodingPractice-04 | 교안 코드 따라치기 - 반복문 캡처, 배열 정렬, List 메서드 (4개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| EventLogger | 이벤트 로거 |
| ArrayProcessor | 배열 처리기 |
| StringFormatter | 문자열 포맷터 |
